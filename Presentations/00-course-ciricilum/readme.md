# Списък на темите

- Изчисляване на NPath и цикломатична сложност
- Стандартни инструменти на Visual Studio за анализ на кода. Тълкуване на резултатите
- Статичен анализ на качеството на C# код с инструмента StyleCop
- Анализиране на компилиран обектен код с FxCop за съответствие с .NET Framework Design Guidelines на Microsoft
- Статичен анализ на кода с платформата Coverity
- Управление на качеството на кода с Git Hooks. Check-in политики
- Continuous integration
- Рефакториране на код

<br>

| :---          |          ---: |

# Конспект за изпит

- Качество на софтуeра. Модели на качеството
- Модел на Боем. Йерархичен модел - фактори, критерии, метрики, оценъчни елементи
- Класификационен модел
- Софтуерни метрики - основни понятия, същност и класификация
- Класически метрики. Метрика за размера на програма. Метрика на Маккейб за структурна (цикломатична) сложност
- Метрика на Холстед за текстуална сложност. Метрика на Рехенберг за технологична сложност. Метрика за четимост
- Стандарти за качеството на софтуера. Управление на качеството
- Инструменти за анализ на кода. Рефакториране.
